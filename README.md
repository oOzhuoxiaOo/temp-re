### setup

使用pnpm安装包，如果没有就先安装pnpm

```cmd
npm i -g pnpm
```

pnpm安装包

```cmd
pnpm i
```

启动运行(开发模式)

```cmd
npm run dev
```

然后就可以在3000端口访问该项目

- Local: http://localhost:3000

## 拥有功能

gltf文件导入导出
json展示和编辑
3d渲染
